# ctw-baseline

Baseline methods for [CTW dataset](https://ctwdataset.github.io/).

## Tutorial

 1. `cd tutorial/`
 2. `jupyter notebook`

You can also preview the tutorial on GitHub: [preview](https://github.com/yuantailing/ctw-baseline/blob/master/tutorial/1-basics.ipynb)
